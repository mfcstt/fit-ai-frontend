## Tarefa

Crie um chatbot que chame a rota '/ia' da API (não use Overal)

## Requisitos técnicos

- Use Vercel IA SDK

const { messages, sendMessage, status } = useChat({
  transport: new DefaultChatTransport({
    api: "/ai",
  }),
});

- Armazene o estado de abertura do chat na URL e uma mensagem inicial que sera enviada automaticamente. Use a biblioteca nuqs para isso.
Exemplo: localhost:3000?chat_open=true?initial_message=message
- Abra o chat pelo botão Sparkeles da nav bar
- Abra o chat ao clicar no botão "?" em Workout Day e a mensagem passada via URL deve ser : "Como executar exercicio X corretamente?"
- Use a biblioteca Streamdown para formatar as mensagens
- Ao abrir o chatbot, caso não haja nenhuma mensagem, exiba a mensagens sugirada: "Moente meu plano de treino" https://www.figma.com/design/r8Iez7UBLVPgfVOwLjJVZx/FIT.AI--Alunos---Copy-?node-id=3606-1037&m=dev e a mensagem padrao: Olá!
